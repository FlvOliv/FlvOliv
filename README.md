### ⭐ Projeto em destaque

#### 🤖 [Malu — Agente de IA no WhatsApp](https://github.com/FlvOliv) <!-- ajuste para o link exato do repositório -->

Chatbot de IA **em produção**, construído e implantado por mim de ponta a ponta, que automatiza o atendimento de uma agência de viagens no WhatsApp — qualifica leads, coleta dados estruturados da viagem e gera briefings prontos para venda antes de repassar para um atendente humano.

- **Arquitetura multi-modelo resiliente** com cadeia de fallback entre provedores (Groq, Cerebras, Mistral, Gemini) + trava de orçamento de tokens → alta disponibilidade com custo quase zero
- **Prompt engineering e guardrails** (sanitização de saída, validação de dados, tratamento de PII com foco em LGPD)
- **Suite de 220+ testes automatizados**; otimizações reduziram a coleta de ~12 para ~7 etapas e cortaram ~50% do custo por cotação
- **Stack:** Python · FastAPI · Redis · deploy em Railway
